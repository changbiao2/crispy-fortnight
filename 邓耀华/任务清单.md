## 三、接口设计（使用Apifox）

### 3.1 需要设计的接口
| 接口名 | 请求方式 | 功能描述 |
|--------|----------|----------|
| register | POST | 用户注册 |
| login | POST | 用户登录 |
| logout | POST | 用户退出登录 |
| resetPassword | PUT | 重置密码 |
| getUserInfo | GET | 获取用户信息 |
| updateUserInfo | PUT | 修改用户信息 |
| getAddressList | GET | 获取收货地址列表 |
| addAddress | POST | 添加收货地址 |
| updateAddress | PUT | 修改收货地址 |
| deleteAddress | DELETE | 删除收货地址 |
| getMessageList | GET | 获取留言列表 |
| postMessage | POST | 发表留言 |
| replyMessage | POST | 回复留言 |
| getChatHistory | GET | 获取聊天记录 |
| sendMessage | POST | 发送聊天消息 |

## 四、数据库设计（使用PowerDesigner）

### 4.1 需要设计的表

| 收货地址表(address) | address_id, user_id, receiver_name, phone, province, city, district, detail_address, is_default |
| 留言表(message) | message_id, user_id, content, parent_id, create_time |
| 聊天记录表(chat_record) | record_id, user_id, admin_id, content, sender_type, create_time |
| 会话表(conversation) | conversation_id, user_id, admin_id, last_message, unread_count, update_time |
