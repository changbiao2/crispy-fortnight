## 三、接口设计（使用Apifox）

### 3.1 需要设计的接口
| 接口名 | 请求方式 | 功能描述 |
|--------|----------|----------|
| listProducts | GET | 获取商品列表 |
| getProductDetail | GET | 获取商品详情 |
| searchProducts | GET | 搜索商品 |
| addToCart | POST | 添加商品到购物车 |
| updateCartQuantity | PUT | 修改购物车商品数量 |
| removeFromCart | DELETE | 从购物车删除商品 |
| getCartList | GET | 获取购物车列表 |
| clearCart | DELETE | 清空购物车 |
